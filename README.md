# drtrack-client

### Prerequisites
* Apache ANT
* JDK 1.8
* Android SDK

### Install
```
$ # Clone the project
$ git clone https://github.com/Powiee/drtrack-client.git
$ # Install dependencies
$ npm install
```

### Run
```
$ # On a emulated device
$ ionic emulate
$ # On a physical device
$ ionic run
